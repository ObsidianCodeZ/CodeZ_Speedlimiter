# CodeZ_Speedlimiter
Simple speed limiter script for your FiveM server

Installation
------------
- Download the file
- Extract the CodeZ_Speedlimtier folder 
- Remove "-main" from file name
- Drop into to your resources folder
- Ensure the resource in your server.cfg
